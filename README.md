# Tech Interview Question Bank

This repository contains beginner-friendly technical interview questions for software and DevOps roles.

## Python Questions
1. What is Python?
2. What are Python lists and tuples?
3. What is a dictionary in Python?

## Git & GitHub Questions
1. What is Git?
2. What is the difference between Git and GitHub?
3. What is a branch?

## DevOps Questions
1. What is DevOps?
2. What is CI/CD?
3. What is Docker?

## Linux Questions
1. What is Linux?
2. What is the `ls` command?
3. What is the `cd` command?

## SQL Questions
1. What is SQL?
2. What is a primary key?
3. What is a join?

More questions will be added soon 🚀
